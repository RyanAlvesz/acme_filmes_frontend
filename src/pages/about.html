Acme Filmes é um projeto desenvolvido por Ryan Alves durante o curso de Desenvolvimento de sistemas do Senai de Jandira . É um projeto full-stack, com o Back-End em Node.js, o banco de dados em MySQL e o Front-End em HTML, CSS, JAVASCRIPT e TailwindCSS. Todo o processo foi orientado pelos professores Marcel Teixeira e Fernando Leonid
